# Mini Booking Engine

Language: TypeScript (Express)

## How to run

API
```bash
cd mini-booking-engine-native/api && npm i && npm run dev
```

Web
```bash
cd mini-booking-engine-native/web && python3 -m http.server 5504
```

Open http://localhost:5504

## Endpoints
- Ping: GET /\n- Bookings: GET /api/bookings
